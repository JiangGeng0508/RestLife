# RestLife

[余生]

这个项目旨在使用Godot mono复原已经停服的众生游。

这是一款人生模拟游戏，你可以在这短暂的一生中体验各种苦乐悲欢。



你无法选择自己出生在什么样的家庭，至少还可以决定自己以后要走什么样的路。

是奋力拼搏，成为人上人。

是闲云野鹤，逍遥人世间。

是合家欢乐，子孙满厅堂。

是血战沙场，马革裹尸还。

还是合纵连横，搅动世界风云。

都由你决定。
